# Session Scope Checklist and Externalization Rules v0.1
UTC_CREATED: 20260306T201237Z

## Session scope checklist
- objective stated
- canon repo target chosen
- pointers/beacons known
- artifacts externalized each wave where possible
- handoff payload externalized
- drain artifact produced before closure

## Rules
- if it is not done programmatically, it is probably not done
- chat is exploration, repo is canon
- human relay should be minimized
- control signals should be thinner than payloads
